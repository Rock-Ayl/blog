# blog项目
------------------------------------
普通的小型个人博客,由于本人目前前后分离，故而web的开发程度不高。


项目简介
----------------------------

开发人员:安永亮 

初始时间:2018-11-1

开发环境：

windows 7 

jdk1.8 

gradle 

开发工具：IEEA 2017， sublime text 3， Hbuilder X

实例地址：www.anyongliang.cn 

服务器： centos_6_09_64

项目集成框架
-----------------------------------
web：

1:JQuery

2:i18n(国际化)

server:

1:springboot

2:quartz（定时器）

3:shiro（权限）

4:Redison（核心功能并发）

数据库
-----------------------------------
Redis：用户缓存

mongoDB: Js的存储

MariaDB: 基于Mysql的关系型数据库,用的是较新的版本，或者至少10.2.1以上
